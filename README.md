# Portfolio Site
This repo contains a portfolio with minimal design and reusable components made with React Typecsript and Tailwind CSS.
This portfolio was heavily inspired by this portfolio of Ted (https://tedawf.com/), whom I came across on a youtube video by Anthony Sistilli (https://www.youtube.com/@AnthonySistilli) from Youtube, where he roasts other dev's portfolios. I took inspiration with the design because it's so clean and eye-pleasing.
